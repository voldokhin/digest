# Смартфоны, банкоматы и "умные" устройства ((https://geektimes.ru/post/287510/ 90% «умных» телевизоров можно взломать вредоносной ТВ-передачей и шпионить за телезрителями)) Вредоносная телепередача позволяет направить встроенный браузер на эксплойт, который получает рутовые права на телевизоре, может установить там произвольный код и использовать ТВ любым образом, как пожелает злоумышленник: начиная со шпионажа и заканчивая проведением DDoS-атак на удалённые цели, когда телевизор становится частью ботнета. Центральным звеном для проведения атаки является технология Hybrid Broadcast Broadband TV (HbbTV) — ТВ-стандарт для передачи дополнительных предложений из интернета на телевизор. По сути, это специальный адаптированный для телевизора сайт, который можно вызвать на экран. Данную технологию поддерживают почти все современные «умные» телевизоры, она совместима с передачей телесигнала по DVB-T, DVB-C и IPTV.
Функцию разблокировки смартфона Samsung Galaxy S8 с помощью лица можно ((https://amp.thehackernews.com/thn/2017/03/samsung-galaxy-s8-facial-unlocking.html обмануть с помощью фотографии)). Для этого достаточно держать фото так, чтобы глаза на изображении были на уровне камеры. https://youtu.be/uS1NmvJvHNk Видео с демонстрацией взлома
Брайан Кребс рассказывает, почему он ((https://krebsonsecurity.com/2017/03/why-i-always-tug-on-the-atm/ всегда внимательно осматривает банкоматы прежде чем воспользоваться ими)). Преступники устанавливают скимминговые устройства для кражи данных карт так, что отличить обычный банкомат от «модернизированного» очень сложно.
Злоумышленники использовали обработку всплывающих диалогов в Safari, чтобы ((https://www.anti-malware.ru/news/2017-03-31/22662 заблокировать браузер пользователя и вымогать деньги)) у своих жертв, требуя подарочные карты iTunes и показывая угрожающие сообщения. Проблема заключалась в том, как Safari обрабатывал всплывающие диалоговые окна одного сайта, затрагивая при этом все приложение, а не только вкладку, на которой был открыт сайт. В версии iOS 10.3, эта ошибка устранена. Блокируя браузер, злоумышленники заставляли пользователей думать, что данные зашифрованы, но любой осведомленный пользователь может просто перейти к настройкам iOS и очистить кэш браузера, чтобы восстановить работоспособность.
Обнаружено ((https://www.pentestpartners.com/blog/vulnerable-wi-fi-dildo-camera-endoscope-yes-really/ множество уязвимостей в «умном» вибраторе Svakom Siime Eye)), который оснащен встроенной камерой и официальным приложением. Устройство имеет видеокамеру и позволяет в режиме реального времени передавать изображение на компьютер, смартфон или планшет по Wi-Fi. Siime Eye работает как точка доступа Wi-Fi, используя одинаковый SSID для всех устройств, причем изменить его нельзя. Пароль по умолчанию уязвим для взлома — «88888888». Таким образом, злоумышленник, находящийся в зоне доступа сигнала, может без труда установить себе мобильное приложение Siime Eye, ввести учетные данные и подключиться к «интимному стриму» (или получить доступ к сделанным ранее видеозаписям и изображениями). https://youtu.be/NdfD4Pod9ho Видео с демонстрацией уязвимого вибратора
Миллионы пользователей смартфонов с чипом Broadcom ((https://googleprojectzero.blogspot.ru/2017/04/over-air-exploiting-broadcoms-wi-fi_4.html уязвимы для взлома по Wi-Fi без участия пользователя)). Хакеры могут эксплуатировать ошибку в коде драйвера беспроводного модуля, которая при определённых условиях приводит к переполнению буфера и выполнению произвольного кода. В число уязвимых устройств входят iPhone, iPad, iPod и Android-смартфоны на базе беспроводного чипа Broadcom. Обновление iOS 10.3.1 устраняет эту уязвимость в устройствах Apple. # Вредоносные программы Мошенники ((https://www.symantec.com/connect/blogs/free-nintendo-switch-emulators-are-fake распространяют вредоносные программы под видом эмулятора новейшей игровой приставки Nintendo Switch)). Под видом «эмулятора» пользователи получают большое количество мусорных программ, наполняющих систему рекламными баннерами.
В рамках атаки на пользователей Skype при запуске домашнего экрана программы отображается рекламный баннер, продвигающий якобы критическое обновление Adobe Flash Player, вместо которого загружается вымогательское ПО. Запуск фальшивого приложения запускает на исполнение обфусцированный код JavaScript, который удаляет только что открытое пользователем приложение и запускает команду PowerShell, загружающую JSE-файл с уже несуществующего домена.
Вирус-вымогатель для Android ((https://www.zscaler.com/blogs/research/new-android-ransomware-bypasses-all-antivirus-programs маскируется под приложение «Одноклассники»). Пользователь может скачать вредоносное ПО в нескольких магазинах приложений, но оно отсутствует в Google Play. Вирус имитирует работу «Одноклассников», но спустя три-четыре часа после установки начинает запрашивать у хозяина устройства права администратора и не дает пользователю спокойно продолжить работу или удалить его. Как только пользователь выдаст нужные права, гаджет заблокируется, а на экране появится сообщение: «Обслуживание вашего устройства приостановлено, а все ваши личные данные перенесены на наш сервер».
# Сайты, мессенджеры и почта В рамках ((http://researchcenter.paloaltonetworks.com/2017/03/unit42-dimnie-hiding-plain-sight/ новой фишинговой кампании для пользователей GitHub)) злоумышленники рассылают электронные письма, инфицирующие системы жертв модульным трояном Dimnie. Письма содержат фальшивые предложения работы, для ознакомления с подробностями которых предлагается открыть вложение. Прикрепленный файл представляет собой архив, после распаковки открывающий Word-документ с макросами. После активирования макросы выполняют ряд команд PowerShell по загрузке и установке трояна Dimnie.
Сбербанк опубликовал на своей странице ВКонтакте ((https://vk.com/page-22522055_52426239?from=wall-22522055_1492409 памятку пользователем мобильного банкинга)) о девяти самых распространённых способах мошенничества.
Обнаружен ((https://threatpost.ru/falshivyj_wordpress-plagin_otkryvaet_bekdor/21282/ фейковый WordPress-плагин для поисковой оптимизации WP-Base-SEO)), открывающий доступ к атакуемому сайту. Вредоносное ПО имитирует легитимный плагин WordPress SEO Tools, и для убедительности снабжено ссылкой на официальную базу плагинов для WordPress и техдокументацию. Код плагина содержит зашифрованный с помощью base64 вызов php-функции eval(), которая выполняет произвольный код PHP, переданный в виде строки, и нередко используется с неблаговидной целью.
Вице-президент компании Barracuda Networks ИБ-эксперт Асаф Сидон (Asaf Cidon) сообщил о ((https://blog.barracuda.com/2017/03/30/threat-spotlight-the-airline-phishing-attack/ фишинговых атаках на клиентов авиакомпаний)). Злоумышленники рассылают сообщения, содержащие либо заражающее систему вредоносное ПО, либо ссылку на подконтрольный им взломанный сайт. Целью киберпреступников является получение личных и корпоративных учетных данных клиентов авиакомпаний.
Китайская хакерская группировка APT10 использует ((http://www.securitylab.ru/news/485797.php новый способ осуществления кибератак)): взламываются сети провайдеров облачных сервисов, которыми пользуется атакуемая организация, после чего с помощью одобренных провайдером каналов связи проникают в корпоративную сеть самой организации. Жертвами APT10 стали организации в Великобритании, США, Индии, Японии, ЮАР, Франции, Бразилии, Канаде, Австралии, Южной Корее и Таиланде. В ходе атак злоумышленники использовали популярное у китайских кибершпионов вредоносное ПО PlugX,а также новый эксклюзивный троян RedLeaves
